Teori

Apa itu Docker Image?

Docker Image adalah template read-only yang berisi:

  • Operating System (misal: Ubuntu, Alpine)
  • Aplikasi kita
  • Dependencies yang dibutuhkan
  • Konfigurasi

Analogi

Image = Resep masakan (tidak berubah)
Container = Masakan yang sudah jadi (bisa dibuat banyak dari 1 resep)

Apa itu Dockerfile?

Dockerfile adalah file teks berisi instruksi untuk membuat Docker Image.

Praktikum

Praktik 1: Aplikasi Python Sederhana

Buat folder project

Terminal
mkdir docker-app
cd docker-app

Buat file app.py

app.py
# app.py
print("=" * 40)
print("Halo! Ini aplikasi Python di Docker")
print("Nama: [Ganti dengan nama Anda]")
print("NIM: [Ganti dengan NIM Anda]")
print("=" * 40)

Buat file Dockerfile

Dockerfile
# Dockerfile
FROM python:3.11-slim

WORKDIR /app

COPY app.py .

CMD ["python", "app.py"]

Build image

Terminal
docker build -t aplikasi-pertama:1.0 .

# Penjelasan:
# -t aplikasi-pertama:1.0 = nama:tag untuk image
# . = lokasi Dockerfile (current directory)

Jalankan container

Terminal
docker run aplikasi-pertama:1.0

Output

========================================
Halo! Ini aplikasi Python di Docker
Nama: [Nama Anda]
NIM: [NIM Anda]
========================================

Dockerfile

Dockerfile
FROM python:3.11-slim

WORKDIR /app

# Copy requirements dulu (untuk caching)
COPY requirements.txt .

# Install dependencies
RUN pip install -r requirements.txt

# Copy source code
COPY app.py .

EXPOSE 5000

CMD ["python", "app.py"]

Build dan Run

Terminal
# Build image
docker build -t flask-app:1.0 .

# Jalankan container
docker run -d -p 5000:5000 --name web-flask flask-app:1.0

# Cek apakah jalan
docker ps
