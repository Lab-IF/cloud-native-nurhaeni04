Teori

Apa itu Docker Compose?

Docker Compose adalah tool untuk mendefinisikan dan menjalankan multiple container menggunakan satu file konfigurasi.

üòì Tanpa Docker Compose

docker run -d --name db mysql
docker run -d --name backend --link db mybackend
docker run -d --name frontend -p 80:80 myfrontend

üéâ Dengan Docker Compose

docker-compose up -d

# Selesai! Semua container jalan!

Struktur docker-compose.yml

docker-compose.yml
version: '3.8'            # Versi format file

services:                  # Daftar container
  web:                     # Nama service
    image: nginx           # Image yang digunakan
    ports:
      - "8080:80"         # Port mapping
  
  database:
    image: mysql
    environment:           # Environment variables
      MYSQL_ROOT_PASSWORD: secret

Perintah Docker Compose

PerintahFungsi
docker-compose upJalankan semua service
docker-compose up -dJalankan di background
docker-compose downStop dan hapus semua
docker-compose psLihat status service
docker-compose logsLihat logs
docker-compose logs -fFollow logs

Praktik 1: WordPress + MySQL

üìÅ wordpress-project/

docker-compose.yml
version: '3.8'

services:
  # Database MySQL
  database:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: rahasia123
      MYSQL_DATABASE: wordpress
      MYSQL_USER: wpuser
      MYSQL_PASSWORD: wppassword
    volumes:
      - db-data:/var/lib/mysql

  # WordPress
  wordpress:
    image: wordpress:latest
    ports:
      - "8080:80"
    environment:
      WORDPRESS_DB_HOST: database
      WORDPRESS_DB_USER: wpuser
      WORDPRESS_DB_PASSWORD: wppassword
      WORDPRESS_DB_NAME: wordpress
    depends_on:
      - database

volumes:
  db-data:
Terminal
# Jalankan
docker-compose up -d

# Lihat status
docker-compose ps

# Lihat logs
docker-compose logs -f

Akses http://localhost:8080 - WordPress siap disetup!
