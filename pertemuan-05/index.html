Naming Convention

Format Nama Image

[username]/[repository-name]:[tag]

Contoh:

  • budiman/my-flask-app:1.0
  • budiman/my-flask-app:latest
  • budiman/my-flask-app:v2

Praktikum

Praktik 1: Buat Akun Docker Hub

Daftar di Docker Hub

Buka https://hub.docker.com dan buat akun gratis

Login dari Terminal

Terminal
docker login

# Masukkan username dan password Docker Hub

Verifikasi Login

Terminal
# Cek kredensial tersimpan
cat ~/.docker/config.json

Praktik 2: Push Image ke Docker Hub

Buat aplikasi sederhana

app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return '''
    <h1>ğŸš€ Halo dari Docker Hub!</h1>
    <p>Nama: [Nama Anda]</p>
    <p>NIM: [NIM Anda]</p>
    <p>Image: [username]/flask-app:1.0</p>
    '''

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)

Buat Dockerfile

Dockerfile
FROM python:3.11-slim
WORKDIR /app
RUN pip install flask
COPY app.py .
EXPOSE 5000
CMD ["python", "app.py"]

Build dengan nama yang benar

Terminal
# Ganti [username] dengan username Docker Hub Anda
docker build -t [username]/flask-app:1.0 .

# Contoh:
docker build -t budiman/flask-app:1.0 .

Push ke Docker Hub

Terminal
docker push [username]/flask-app:1.0

# Contoh:
docker push budiman/flask-app:1.0

Berhasil!

Image sekarang ada di Docker Hub! Buka hub.docker.com/r/[username]/flask-app

Praktik 3: Pull Image dari Docker Hub

Terminal
# Hapus image lokal dulu (simulasi di komputer lain)
docker rmi [username]/flask-app:1.0

# Pull dari Docker Hub
docker pull [username]/flask-app:1.0

# Jalankan
docker run -d -p 5000:5000 [username]/flask-app:1.0

Praktik 4: Update dan Versioning

Terminal
# Update app.py dengan fitur baru...

# Build versi baru
docker build -t [username]/flask-app:2.0 .

# Push versi baru
docker push [username]/flask-app:2.0

# Tag sebagai latest
docker tag [username]/flask-app:2.0 [username]/flask-app:latest
docker push [username]/flask-app:latest

Tugas Praktikum

Tugas: Publish Aplikasi ke Docker Hub 100 poin

  1. Buat akun Docker Hub (jika belum punya)
  2. Buat aplikasi Flask dengan fitur:
    • Halaman biodata (/)
    • Halaman timestamp (/waktu) menampilkan waktu sekarang
  3. Build image dengan nama: [dockerhub-username]/biodata-[nim]:1.0
  4. Push ke Docker Hub
  5. Screenshot halaman repository di Docker Hub
  6. Berikan link Docker Hub ke dosen

Penting

Jangan lupa docker logout setelah selesai jika menggunakan komputer lab!
