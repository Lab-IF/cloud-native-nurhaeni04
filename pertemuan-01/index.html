Teori

Apa itu Docker?

Docker adalah platform yang memudahkan kita untuk membuat, menjalankan, dan mendistribusikan aplikasi dalam container.

Analogi Sederhana

Container = Kotak/Box yang berisi aplikasi + semua yang dibutuhkan untuk jalan.
Seperti "packaging" makanan - sudah lengkap dan siap dimakan di mana saja!

Virtual Machine

  • Berat (GB)
  • Lambat start (menit)
  • Butuh full OS setiap VM
  • Resource intensive

Container

  • Ringan (MB)
  • Cepat start (detik)
  • Share OS kernel
  • Efisien resource

Istilah Penting

Image = Template/blueprint (seperti class di OOP)
Container = Instance yang berjalan (seperti object di OOP)
Registry = Tempat menyimpan images (seperti GitHub untuk code)

Instalasi Docker

Install Docker

Terminal 
# Update package
sudo apt update

# Install Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# Tambahkan user ke group docker
sudo usermod -aG docker $USER

Verifikasi Instalasi

Terminal 
# Cek versi Docker
docker --version

# Output: Docker version 24.0.7, build afdd53b

Test Docker

Terminal 
# Jalankan hello-world
docker run hello-world

Output yang Diharapkan

Hello from Docker! This message shows that your installation appears to be working correctly.

Praktikum

Praktik 1: Menjalankan Container Pertama

Terminal 
# Jalankan container nginx (web server)
docker run -d -p 8080:80 --name web-pertama nginx

# Penjelasan:
# -d          = detached mode (jalan di background)
# -p 8080:80  = port mapping (akses via localhost:8080)
# --name      = beri nama container
# nginx       = nama image

Buka Browser

Akses http://localhost:8080 - Anda akan melihat halaman "Welcome to nginx!"

Praktik 2: Perintah Docker Dasar

Terminal 
# Lihat container yang berjalan
docker ps

# Lihat semua container (termasuk yang stop)
docker ps -a

# Lihat logs container
docker logs web-pertama

# Stop container
docker stop web-pertama

# Start container lagi
docker start web-pertama

# Hapus container (harus stop dulu)
docker stop web-pertama
docker rm web-pertama

Praktik 3: Container Interaktif

Terminal 
# Jalankan Ubuntu dan masuk ke dalamnya
docker run -it ubuntu bash

# Sekarang Anda di dalam container Ubuntu!
# Coba jalankan:
cat /etc/os-release
ls
pwd

# Keluar dari container
exit

Rangkuman Perintah

Perintah Fungsi
docker run <image> Jalankan container dari image
docker ps Lihat container yang berjalan
docker ps -a Lihat semua container
docker stop <name> Stop container
docker start <name> Start container
docker rm <name> Hapus container
docker images Lihat semua images
docker pull <image> Download image
docker logs <name> Lihat logs container
